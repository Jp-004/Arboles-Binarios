# Arboles-Binarios
Repositorio dedicado a la práctica e implementación de Estructuras de Datos y Algoritmos en Java. Incluye el desarrollo desde cero de Árboles Binarios, Heaps, Colas de Prioridad y algoritmos recursivos, con un fuerte enfoque en la eficiencia, el diseño orientado a objetos y la validación mediante tests automatizados.
